开发者工具验证

1. 按 F12 打开开发者工具

2. 切换到 Console 标签

3. 粘贴并运行以下代码：

// 验证滚轮事件
const svg = document.querySelector('.topology-graph svg');
if (svg) {
    console.log('✓ SVG 找到');
    console.log('✓ onWheel 绑定:', svg.onwheel !== null);
} else {
    console.log('✗ SVG 未找到');
}

// 验证节点 tooltip
const nodes = document.querySelectorAll('.graph-node-group title');
if (nodes.length > 0) {
    console.log('✓ 找到', nodes.length, '个节点');
    console.log('✓ 第一个节点 tooltip:');
    console.log(nodes[0].textContent);
} else {
    console.log('✗ 未找到节点');
}
